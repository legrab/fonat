# Content authoring

Use Hungarian as the canonical language for the reference material. Store educational text as Markdown with LaTeX math. Keep source and rights metadata on every externally derived item.

Authoring should start with the educational content. Add only enough metadata to make the item discoverable and reusable. Concept suggestions are deterministic and optional.

## Rights states

- project-owned
- teacher-owned
- openly-licensed
- permission-granted
- public-domain
- private-use-only
- unknown

Public packages must not publish private-use-only or unresolved material.

## External AI

Export an AI task bundle from the Administration page. The bundle includes selected concepts, existing exercises, contract expectations, and instructions to ask the teacher a small number of focused questions before generating a valid package.
